# Capstone Project: Vulnerable Booking Application

**CSC 489 - Web Application Security | Spring 2026**

## Prerequisites

- Node.js 18+
- Python 3.8+
- uv (recommended for macOS, Linux, and Windows)

## Setup and Run Instructions

### macOS/Linux: Option A (.sh) OR Option B (Makefile)

#### Option A (.sh)

```bash
chmod +x start-dev.sh
./start-dev.sh
```

OR

#### Option B (Makefile)

```bash
make dev
```

### Windows (.bat)

```bat
start-dev.bat
```

## App URLs

- Frontend: http://localhost:3001
- Backend API: http://localhost:5001
